# PySMAD: Pythonic Spacecraft Mission Analysis and Design

This software suit is designed to empower scientists, researchers, and engineers to create, simulate, and validate spacecraft designs. In its final form, this software suite will include tools for 

- orbit propagation, spaceflight mechanics, and astrodynamics
  - attitude and orbit control
- spacecraft thermal design and analysis
- spacecraft power budget and mission mode analysis
- spacecraft link budget and communications analysis
